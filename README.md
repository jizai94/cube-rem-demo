# cube-rem-demo

> a cube-ui demo with rem layout

## Build Setup

``` bash
# install dependencies（安装依赖）
npm install

# serve with hot reload at localhost:8080（开发环境）
npm run dev

# build for production with minification（打包程序）
npm run build

# build for production and view the bundle analyzer report（打包程序同时打印包的分析报告）
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```

## directory

``` directory
build -- 编译任务的代码
config -- webpack 的配置文件
node_modules -- 依赖的模块集
src -- 编辑代码处
static -- 静态资源目录
test -- 测试代码处
.babelrc -- babel的默认配置
.editorconfig -- 编辑器配置
.eslintignore -- eslint代码检查的忽略目录
.eslintrc.js -- eslint的默认配置
.gitignore -- git版本管理忽略目录及文件
.postcssrc.js -- postcss（px2rem/px2vw）的相关配置
index.html -- html模板
package.json -- 项目的基本信息
README.md -- 项目介绍及相关描述
```
